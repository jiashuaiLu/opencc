### 🏗️ 项目结构

```
dongcc/
├── src/
│   ├── main/                    # 主进程 ✅
│   │   ├── index.ts            # 主进程入口
│   │   ├── preload.ts          # Preload 脚本
│   │   ├── proxy/              # 代理服务
│   │   │   └── server.ts       # 代理服务器
│   │   ├── database/           # 数据库
│   │   │   └── index.ts        # 数据库管理
│   │   ├── logger/             # 日志系统
│   │   │   └── index.ts        # 日志管理
│   │   ├── ipc/                # IPC 通信
│   │   │   └── handlers.ts     # IPC 处理器
│   │   └── system/             # 系统检查
│   │       └── checker.ts      # 环境检查器
│   ├── renderer/               # 渲染进程 ✅
│   │   ├── components/         # 组件
│   │   │   └── Layout.tsx      # 布局组件
│   │   ├── pages/              # 页面
│   │   │   ├── Dashboard.tsx   # 仪表盘
│   │   │   ├── Config.tsx      # 配置管理
│   │   │   ├── Logs.tsx        # 日志查看
│   │   │   ├── History.tsx     # 对话历史
│   │   │   └── Settings.tsx    # 系统设置
│   │   ├── types/              # 类型定义
│   │   │   └── electron.d.ts   # Electron API 类型
│   │   ├── App.tsx             # 应用根组件
│   │   ├── index.tsx           # 渲染进程入口
│   │   └── styles/             # 样式
│   │       └── global.css      # 全局样式
│   └── shared/                 # 共享代码 ✅
│       └── types.ts            # 类型定义
├── package.json                # 项目配置 ✅
├── tsconfig.json               # TypeScript 配置 ✅
├── tsconfig.main.json          # 主进程 TS 配置 ✅
├── vite.config.ts              # Vite 配置 ✅
├── .eslintrc.json              # ESLint 配置 ✅
├── .prettierrc                 # Prettier 配置 ✅
├── index.html                  # HTML 入口 ✅
├── dev.sh                      # 开发启动脚本 ✅
└── README.md                   # 项目文档 ✅
```

### 🚀 如何运行

#### 1. 安装依赖
```bash
cd dongcc
npm install
```

#### 2. 开发模式
```bash
npm run electron:dev
```

#### 3. 构建应用
```bash
npm run electron:build
```
